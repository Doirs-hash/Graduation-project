### 工厂订单管理系统
1.factory为前端文件，采用react+antd技术


1. 进入factory文件内 **npm i** 下载包文件
2. **npm start**运行


2.factoryOrderProject为后端文件，采用node.js+mysql

1. 进入factoryOrderProject文件夹输入指令 **npm i** 下载包文件
2. **node server**运行
